# Shopping Management System

## Overview

A terminal-based Shopping Management System built using Python and MySQL. The project supports user authentication, shopping cart management, order processing, and an admin dashboard.

---

## Features

### Customer
- Register
- Login / Logout
- View Products
- Search Products
- Add to Cart
- View Cart
- Remove Items
- Checkout
- View Order History

### Admin
- Login
- View Products
- Add Product
- Update Product
- Delete Product
- Dashboard

---
